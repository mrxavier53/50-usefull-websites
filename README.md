# 50 Useful Websites

A simple animated webpage with 50 useful websites, sorted into categories like Productivity, AI Tools, Design, Development, and Learning.

Made by **Mr Xavier**.

## What's in this folder

- `50-usefull-websites.html` → the main page (open this!)
- `open.bat` → double-click to open the page in your browser
- `serve.bat` → double-click to run it on a local server
- `INSTRUCTIONS.md` → how to use and edit this project
- `LICENSE` → usage rules / copyright info

## How to open it

Just double-click `50-usefull-websites.html` or `open.bat`. That's it!
